# Syndeo

Multi-channel conversational AI management platform.

## Project Structure

```
prototype/
  pages/        # HTML page prototypes
  components/   # Reusable HTML/CSS components
  assets/
    images/     # Logos, illustrations
    icons/      # Icon assets
    fonts/      # Custom fonts
  styles.css    # Global styles & design tokens

design/
  tokens/       # Colour, spacing, typography reference
  guidelines/   # Design rules and patterns

references/     # Local-only reference PDFs (gitignored)
```
